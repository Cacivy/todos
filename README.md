# ToDos

一个集待办事项和活动于一身的高度可定制化todo类产品。

### 需求

+ 待办事项
  + 记事
    + 文字
    + 图文
    + 链接
  + 提醒
    + 时间
    + 地点
  + 子任务
    + 只有两个层级
  + 星标
    + 置顶
  + 归档
+ 日程
  + 地图
    + 路线
  + 天气
    + 日历中当前月显示
    + 提前提醒
  + 邀请
    + Email/手机号
